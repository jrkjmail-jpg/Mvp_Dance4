# Танцуй — Oversized Typography MVP

Интерактивный веб-клиент платформы для танцевальных студий, педагогов и учеников.

## Возможности

- два режима: педагог и ученик;
- постоянное пространство студии и группы;
- создание групп с сохранением в `localStorage`;
- подготовка урока из двух независимых видео;
- демонстрационный пайплайн построения pose-эталона;
- список учеников и прогресс;
- обучение, сдача с отсчетом и экран результата;
- адаптивный интерфейс в стиле Oversized Typography.

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
```

## Важно

Текущая версия — полноценный интерактивный frontend-прототип. Pose-анализ и серверное хранение представлены интерфейсными сценариями и подготовленными границами интеграции; для production необходимо подключить MediaPipe, backend, S3-совместимое файловое хранилище и авторизацию согласно исходной спецификации проекта.
